# Детектор запрещенных организаций

## Требования
.NET 8.0 SDK

## Установка
```powershell
git clone https://github.com/murlinmurlo/organization-checker.git
cd organization-checker
```

## Запуск
```
dotnet run
```

##  Используемые алгоритмы

**1. Алгоритм Ахо-Корасик**

Оптимален для большого количества паттернов

**Особенности:**

Построение префиксного дерева
Failure links для быстрых переходов
Линейное время поиска O(n + m + z)

**Сложность:** O(n + m + z),
n - общая длина текста (сумма длин всех строк)
m - суммарная длина всех паттернов (всех запрещенных организаций)
z - общее количество найденных вхождений


**2. Инвертированный индекс**

**Особенности:**

Построение индекса по словам

Быстрая фильтрация кандидатов

**Сложность:** O(k * log n) для поиска кандидатов,
k - количество уникальных слов в запросе
n - количество документов в индексе
log n - логарифмическая сложность поиска в индексе

**3. SIMD-ускоренный поиск**

**Особенности:**

Использование AVX2/SSE2 инструкций

**Сложность:** O(k × n/m) для SIMD операций,
n - длина текста
m - длина вектора SIMD (16 или 32 байта)
k - количество паттернов
